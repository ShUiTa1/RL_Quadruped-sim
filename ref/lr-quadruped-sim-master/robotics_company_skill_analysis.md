# 机器人公司技能需求分析：按产品、产出、招聘岗位与技术栈拆解

## 总结判断

这些公司并不是都在招同一种“机器人技能”。真正适合你“控制 + 动力学 + MPC/RL + 未来补 VLA/模仿学习”的公司，主要可以分成 5 类：

1. **本体/人形核心公司**：宇树、智元、傅里叶、银河通用、众擎、星尘、乐聚、Sharpa、帕西尼、自变量。  
   核心看：运动控制、全身控制、RL、仿真到真机、遥操作数据、VLA/模仿学习、灵巧操作。

2. **大模型/互联网切入具身智能**：字节 Seed、阿里达摩院、蚂蚁灵波、商汤、美的 AI 研究院。  
   核心看：VLA/VLM、多模态大模型、数据闭环、模仿学习、强化学习后训练、机器人操作模型。

3. **平台/基础设施型**：英伟达、腾讯 Robotics X。  
   核心看：仿真平台、Isaac/Omniverse、机器人学习框架、Sim2Real、系统工程、算法平台化。

4. **物流/清洁/家电/导航应用型**：京东、追觅、TP-Link、高德。  
   核心看：SLAM、导航、路径规划、运动控制、工程落地。对 VLA/RL 有需求，但一般不是最前沿核心。

5. **视觉/算法公司外延机器人**：商汤、部分阿里/美的岗位。  
   核心看：多模态感知、世界模型、VLA、自动驾驶式数据闭环；但真实机器人控制深度要看具体团队。
